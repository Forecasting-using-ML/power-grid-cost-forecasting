# Raw Data

This directory contains subpackages to download and extract raw data from
different sources.

## Hourly Price Data

The [hourly-price](hourly-price) package sanitizes a single curve of hourly
price data ($/MWHr) for the state of New York.

## Fuel Output Data

The [fuel-output](fuel-output) package sanitizes a multivariate data of fuel
output data (MWHr) for the state of New York.
